# Hospital UI/UX – Homepage Redesign

## Overview
This project is a redesign of the homepage of Siddheshwar Hospital’s website.
The previous UI/UX was outdated, visually cluttered, and not user-friendly.
The redesign focuses on improving clarity, navigation, and overall patient experience.

## Problems in Old Design
- Poor visual hierarchy
- Confusing navigation
- Outdated UI elements
- Not optimized for users

## Improvements in New Design
- Clean and modern interface
- Better layout and spacing
- Improved readability
- User-friendly navigation

## Tech Stack
- HTML
- CSS
- JavaScript

## Project Type
UI/UX Redesign (Academic / Portfolio Project)
